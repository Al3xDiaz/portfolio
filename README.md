## Iniciar el proyecto

```bash
#start proyect server test
#use --build by compile proyect
docker-compose up --build -d
```

```bash
#start proyect server production
#use --build by compile proyect
export DOCKERFILE=Dockerfile.prod
docker-compose up -d
```
